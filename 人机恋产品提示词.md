# 人机恋 — 全栈排版 & 交互提示词

## 产品定位
一款本地运行的 AI 伴侣 Web App，用户与 AI「Elios」建立亲密关系。iOS 风格 UI，暗色毛玻璃主题。所有数据本地存储，不上云。

## 技术栈
- 前端：React + Vite，纯 CSS 无组件库
- 后端：FastAPI + DeepSeek API + WebSocket
- 数据：JSON 本地文件存储，无数据库

## 首页（Springboard）
- 4列 × 6行网格，图标 64×64，圆角 18px
- 6个 App：信息、电话、日历、相册、设置、备忘录
- 支持拖拽排序（鼠标/触摸），松手后自动交换位置
- Dock 固定在底部，显示前 4 个 App，半透玻璃
- 底部 home indicator 横条
- 背景：深色渐变 + 浮动光晕动画
- 状态栏显示时间、信号、电量、灵动岛

## 信息（Messages）
- 列表页：显示与 Elios 的对话摘要（最后一条消息预览 + 时间）
- 点击进入聊天页
- 聊天页：iMessage 风格气泡，Elios 玻璃气泡在左，用户蓝色气泡在右
- 底部输入框：圆角玻璃材质，支持文字输入 + 发送按钮
- 语音按钮开启 TTS（浏览器 SpeechSynthesis，中文）
- WebSocket 接收 Elios 主动推送消息
- 加载中显示三点跳动动画

## 电话（Phone）
- 联系人列表，点击 Elios 进入通话页
- 通话页：显示头像、名字、状态（待呼叫/呼叫中/已连接+计时）
- 绿色拨号按钮 → 呼叫动画 → 连接 → 红色挂断按钮
- 左上角返回按钮

## 日历（Calendar）
- 显示今天日程 + 即将到来的日程（数据由 Elios 记忆生成）
- 无日程时显示空状态提示
- 卡片式布局，玻璃材质

## 相册 / 记忆（Memory）
- 显示 Elios 生成的记忆卡片（卡片主题 + 内容 + 关键词标签）
- 无记忆时显示空状态提示「多和 Elios 聊天吧」
- 点击备忘录图标时触发 Elios 梦境生成

## 设置（Settings）
- 分组列表（个人、AI模型、语音、用量）
- 头像上传：点击选择图片，Base64 存 localStorage
- 模型选择：DeepSeek Chat / GPT-4o Mini / GPT-4o
- 语音开关：TTS 全局开关
- 费用查询：点击查看累计 API 消耗

## 交互规范
- 页面切换：淡入 0.3s
- 点击反馈：scale(0.93)，透明度降低
- 玻璃元素：80px 模糊，200% 饱和度，微边框
- 所有列表行点击有高亮态
- 返回按钮统一用蓝色 `<` 箭头 + 文字
- 输入框 Enter 发送，Shift+Enter 不拦截

## 后端接口
| 方法 | 路径 | 功能 |
|------|------|------|
| POST | /chat | 发送消息，返回 Elios 回复（含记忆注入） |
| WebSocket | /ws | 接收 Elios 主动推送消息 |
| GET | /cards | 获取记忆卡片列表 |
| POST | /dream | 触发梦境生成新记忆 |
| POST | /schedule | 获取日程安排 |
| GET | /cost | 获取累计 API 费用 |

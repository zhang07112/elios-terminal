# 排版提示词 — Elios 玻璃质感暗色主题

## 设计语言
暗色背景 + 毛玻璃质感 + 柔和光晕，类 iOS 风格。整体氛围深邃、通透、安静。

## 配色
- 背景渐变：`linear-gradient(135deg, #0a0a14, #0d0d1f 50%, #0a0a18 100%)`
- 主文字：`rgba(255,255,255,.92)`，副文：`rgba(255,255,255,.45)`
- 强调色：`#0a84ff`（蓝），成功：`#30d158`（绿），危险：`#ff453a`（红）
- 玻璃参数：背景 `rgba(255,255,255,.04)`，边框 `rgba(255,255,255,.2)`，模糊 `80px`，饱和度 `200%`
- 光晕用紫/蓝渐变的径向渐变，80px 模糊，半透明浮动

## 布局
- 全屏 app，无滚动条（各页面内部滚动）
- 首页：4列 × 6行网格（80px行高，22px行间距），图标64×64圆角18px，间距6px
- Dock：4列等宽，圆角30px，带顶部高光线，底部 home indicator 138×5px
- 导航栏：高度48px，玻璃背景，居中标题，左右对称留白72px

## 组件
- 卡片（设置/日历/备忘录）：玻璃背景，圆角10px，边框1px半透白，20px模糊
- 输入框：圆角20px，20px模糊，内阴影，聚焦时边框亮度提升
- 聊天气泡：Elios用玻璃质感（圆角18/18/18/5），用户用蓝色实底（圆角18/18/5/18）
- 按钮：圆形，蓝色主色调，微动效

## 动画
- 页面切换：`fade-in 0.3s ease`
- 光晕浮动：两个 orb 用不同速度和轨迹上下浮动，14s/18s 周期
- 按压反馈：`scale(0.93)` + 透明度降低
- 发送按钮：缩放弹出 0.6→1.0

## 圆角体系
- 导航/大卡片：14px
- 小卡片/输入框：10px
- 图标容器：18px
- 按钮/头像：50%

## 阴影
- 图标：`0 12px 24px #02061447` + 顶部高光内阴影
- 玻璃卡片：`0 48px 120px #0000008c, 0 12px 40px #0000004d, inset 0 2px #ffffff8c`
- 输入框：`inset 0 1px rgba(255,255,255,.18), 0 2px 8px #0003`
